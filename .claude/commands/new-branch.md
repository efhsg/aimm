# New Branch Command

Create a new feature or fix branch following project naming conventions.

## Instructions

Load and follow the skill at `.claude/skills/new-branch.md`.

## Quick Reference

Branch types:
- `feature` — New functionality
- `fix` — Bug fix
- `refactor` — Code restructuring
- `chore` — Maintenance tasks

Format: `{type}/{description}` (e.g., `feature/add-macro-collection`)
